# 📊 Loan Risk Prediction Dataset (100K Records)

## 📌 Project Overview
This project contains a dataset of **100,000 loan applications** used for analyzing and predicting loan default risk.  
It can be used for **Machine Learning classification tasks**, especially in financial risk modeling.

---

## 📂 Dataset Information

- **File Name:** `industry_loan_risk_dataset_100k.csv`
- **Size:** 100,000 records
- **Type:** Structured tabular data
- **Domain:** Finance / Banking / Risk Analytics

---

## 🎯 Objective
The main goal of this dataset is to help build models that can:

- Predict loan default risk
- Analyze customer credit behavior
- Support financial decision-making
- Improve lending strategies

---

## 🧾 Possible Features (Typical Columns)
*(Actual columns may vary depending on dataset)*

- Customer ID
- Age
- Income
- Employment Type
- Loan Amount
- Credit Score
- Loan Purpose
- Debt-to-Income Ratio
- Previous Defaults
- Loan Status (Target Variable)

---

## 🤖 Machine Learning Use Cases

You can use this dataset for:

- Neural Networks
